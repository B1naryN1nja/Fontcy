# Fontcy
Fontcy is a simple fancy text generator for social media, websites, or anywhere that doesn’t support custom fonts. It uses Unicode characters behind the scenes and is completely open source. Other “fancy text” generators exist, but they’re often cluttered, poorly designed, filled with ads, or even locked behind paywalls.

As someone who uses Unicode styling for account bios, social media, and various other places on the internet, I got tired of searching for something that suited my needs—so I made my own.

Fontcy is built entirely for the modern web using HTML, CSS, and JavaScript. I also plan to create native apps for macOS, iOS, Windows, and Android in the near future. Contributions are 100% welcome, and if there’s a “fancy” font you’d like to see, feel free to open an issue!

# Core Features

- Ad free, always free
- Tons of fonts, but none that are broken, or simply unnecessary.
